nginx build说明
===============

如何新增虚拟机配置
------------------

1. 一种是在build目录下面的conf.d中新增虚拟主机配置，然后重启nginx服务

2. 将nginx的配置文件挂载到容器内部，修改或新增虚拟机配置后重启nginx服务


配置文件位置
------------

主配置文件 /etc/nginx/nginx.conf ,额外配置目录为 /etc/nginx/conf.d

日志文件
---------

日志默认保存在 /var/log/nginx 中，可根据需要挂载到本地目录，方便查看
